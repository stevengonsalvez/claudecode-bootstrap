# Usage Guide

## Invoking the Skill

The skill activates automatically when you mention cryptocurrency:
- "What's Bitcoin doing?"
- "Analyze Ethereum"
- "Crypto market overview"

## Modes

### Lightweight (Default)
- 4 haiku agents
- 30-60 seconds
- Good for quick checks

### Comprehensive  
- 12 agents (haiku, sonnet, opus)
- 2-5 minutes
- Deep multi-perspective analysis

### Silent
- Files only, no display
- For automation/background work

## Specifying Cryptocurrencies

Mention ticker symbols:
- "Bitcoin" or "BTC"
- "Ethereum" or "ETH"
- "Solana" or "SOL"

## Output

Results saved to `outputs/YYYY-MM-DD_HH-MM-SS/`:
- `crypto_market/` - Market analysis
- `crypto_analysis/` - Coin-specific
- `crypto_macro/` - Macro correlations
- `crypto_plays/` - Investment opportunities

## Tips

1. Start with lightweight mode
2. Upgrade to comprehensive for big decisions
3. Check timestamps for data freshness
4. Review multiple agent perspectives
